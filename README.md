# CBAA
Component Based Angular App

Requirements:
1. nodejs. Install it in any way you like, but I recommend to use nvm https://github.com/creationix/nvm.
2. git client https://goo.gl/JMYbqv.

To start application:
1. Clone this repos.
2. Install node_modules.
3. Run npm start.

http://localhost:8080
